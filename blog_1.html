In your reflections, answer the following questions:

Whats the difference between Git and GitHub? git is the software/code that when used, means multiple people (in a team, for example) can collaborate on the same code without overwriting each other's work. Git has flexible parameters for how these edits interact with each other. Github is a service which acts as a third party host for people using Git to collaborate on code. This means users can work together remotely, and it has social network elements where people can follow each other and upload/share their coding projects.

Can you think of an analogy to describe them? git is Elton John and Bernie Taupin writing songs together. Github is them doing it over the phone - but they're both talking to an operator? And the operator has a bunch of notes they're keeping track of and feeding and receiving and compiling different versions of verses and choruses etc into the one song.
